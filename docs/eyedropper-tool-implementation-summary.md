## 🛠 技术实现

### 核心技术栈
- **前端框架**: Next.js 15 + React
- **样式方案**: Tailwind CSS
- **颜色处理**: 项目内置colorConverter工具
- **状态管理**: React Hooks
- **存储方案**: localStorage
- **通知系统**: Sonner Toast

### API集成
- **EyeDropper API**: 浏览器原生颜色拾取
- **支持浏览器**: Chrome 95+, Edge 95+, 其他Chromium内核
